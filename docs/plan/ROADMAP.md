## 项目概述

BlockType 是一个从零开发的 **C++26 标准编译器前端**，后端使用 LLVM，核心特色：

| 特色 | 说明 |
|------|------|
| C++26 标准 | 支持反射、Contracts、#embed 等新特性 |
| 中英双语编程 | 原生支持中文和英文关键字、标识符、诊断消息 |
| AI 驱动 | 深度集成 AI 能力，提供智能诊断和优化建议 |
| LLVM 后端 | 使用 LLVM 17+ 生成高质量目标代码 |
